Vedremo i principali elementi del linguaggio HTML

Le pagine Web

Le pagine Web sono costruite mediante l'utilizzo di due elementi principali:
  • Linguaggio HTML: contenuto della pagina Web
  • Fogli di stile CSS: layout
Le pagine Web contengono due tipi di oggetti:
  • Testo del documento
  • Marcatori del linguaggio HTML: tag

Regole fondamentali per la corretta scrittura delle pagine Web

  1. I tag sono scritti tra parentesi angolari
  2. Ogni tag deve essere aperto e chiuso
  3. Normalmente la chiusura di un tag avviene tramite la barra / (slash) che precede il nome del tag
  4. I file di una pagina Web hanno estensione .htm o .html

Struttura generale di una pagina Web

<html>

<head>

</head>


<body>

</body>


</html>
I due tag <head> e <body> si trovano sempre all'interno del tag <html>.
Il tag </html> è chiamato tag radice in quanto contiene al suo interno tutti gli altri tag.

HEAD

La testa (head) contiene informazioni che nella maggior parte dei casi non vengono visualizzate direttamente. Gli elementi principali che si utilizzano sono:
  • Tag title: contiene il titolo della pagina che verrà visualizzato nella barra del titolo del browser
  • Metadati: informazioni invisibili agli utenti, ma utili per la classificazione della pagina Web e importanti per l'indicizzazione corretta all'interno dei motori di ricerca
    • charset: definisce l'elenco dei caratteri che vengono usati
    • author
    • description
    • keywords

BODY

All'interno del body si trova il testo vero e proprio, comprensivo di evenutali link immagini e tabelle.
